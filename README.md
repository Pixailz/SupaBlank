# SupaMakefile
Lib of mk file for a Fancy makefile

## features

1. lib splitted in [mk](./mk) folder
1. info bar for objs ([pb.mk](./mk/pb.mk))
   1. little loading bar with braille unicode char
   1. display index
   1. eslapsed time on the right of the screen
